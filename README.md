Dotfiles
---

### Quick installation:
Just paste this into a terminal:

    curl -s https://raw.githubusercontent.com/uridium/dotfiles/master/install.sh | bash

### Manual configuration:
    git clone git@github.com:uridium/dotfiles.git

##### Install:

    cd dotfiles
    ./setup.sh install

##### Restore:

    cd dotfiles
    ./setup.sh restore
