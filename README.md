Dotfiles
---

### Quick installation:
Just paste this into a terminal:

    curl -s https://raw.githubusercontent.com/radar-aol-pl/dotfiles/master/install.sh | bash

### Manual configuration:
##### Install:

    git clone https://github.com/radar-aol-pl/dotfiles.git
    cd dotfiles
    ./setup.sh install

##### Restore:

    cd dotfiles
    ./setup.sh restore
